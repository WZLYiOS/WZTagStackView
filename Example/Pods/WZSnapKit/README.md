# 我主良缘SnapKi框架库扩展

## Requirements:
- **iOS** 9.0+
- Xcode 10.0+
- Swift 5.0+


## Installation Cocoapods
<pre><code class="ruby language-ruby">pod 'WZSnapKi', '~> 2.0.0'</code></pre>

## Instructions
- 因为最新的SnapKi支持Swift5.0,但是最低iOS系统要求是10开始，目前公司没办法直接从10开始，所以这边把官方的4.2.0下载下来然后做修改让其支持Swift5.0
